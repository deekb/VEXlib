# Installation

## Support
VEXlib officially supports the following platforms:
- Linux (x86_64 and ARM)
- Windows (32-bit and 64-bit)

> Please note that macOS is currently untested, [DM me](https://discord.com/users/892894045439946762) if you would like help/support.  I don't have access to a machine with macOS on it, and it would be challenging to fully test it using a VM 
>
{style="note"}

## Installation

To install VEXlib, clone the GitHub repository:


<tabs>
<tab title="Using git">
<code-block lang="bash">
git clone https://github.com/deekb/VEXlib.git
</code-block>
</tab>
<tab title="Without git">
You can download the zip version of the code <a href="https://github.com/deekb/VEXlib/archive/refs/heads/master.zip">Here</a> and extract it
</tab>
</tabs>

Once the code is downloaded you should open it in your IDE of choice and configure it
