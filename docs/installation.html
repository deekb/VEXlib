VEXlib 1.0 Help

Installation

Support

VEXlib officially supports the following platforms:

  • Linux (x86_64 and ARM)

  • Windows (32-bit and 64-bit)

Please note that macOS is currently untested, DM me if you would like help/support. I don't have access to a machine with macOS on it, and it would be challenging to fully test it using a VM

Installation

To install VEXlib, clone the GitHub repository:

git clone https://github.com/deekb/VEXlib.git

You can download the zip version of the code Here and extract it

Once the code is downloaded you should open it in your IDE of choice and configure it
